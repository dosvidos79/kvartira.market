# Квартира.Market

Сервис аренды недвижимости, где в центре не квартиры, а арендаторы.

## Концепция

Платформа, где люди, ищущие жильё, создают профили-резюме с информацией о себе, работе, доходе и предпочтениях. Собственники квартир просматривают базу таких профилей и сами предлагают жильё подходящим арендаторам.

## Основные функции

- 📋 Профили арендаторов с фото, работой, доходом, предпочтениями
- ⭐ Система отзывов от прошлых арендодателей
- ✅ Верификация личности и дохода
- 🏆 Рейтинг надёжности арендаторов
- 💬 Встроенный чат для общения
- 🔒 Контакты не раскрываются до начала общения
- 💳 Система подписок для собственников
- 🚀 Платная верификация и продвижение профилей

## Технологии

- Next.js 14 (App Router)
- TypeScript
- Prisma (ORM)
- NextAuth (аутентификация)
- Tailwind CSS
- Radix UI

## Установка

```bash
npm install
```

## Настройка базы данных

1. Создайте файл `.env` в корне проекта:
```
DATABASE_URL="postgresql://user:password@localhost:5432/kvartira_market"
NEXTAUTH_SECRET="your-secret-key"
NEXTAUTH_URL="http://localhost:3000"
```

2. Запустите миграции:
```bash
npm run db:push
npm run db:generate
```

## Запуск

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Основные страницы

### Для арендаторов:
- `/dashboard` - Главная панель
- `/dashboard/profile` - Редактирование профиля
- `/dashboard/offers` - Предложения от собственников
- `/dashboard/messages` - Встроенный чат

### Для собственников:
- `/dashboard` - Главная панель
- `/dashboard/tenants` - База арендаторов
- `/dashboard/tenants/[id]` - Профиль арендатора
- `/dashboard/properties/new` - Добавить квартиру
- `/dashboard/subscription` - Управление подпиской
- `/dashboard/messages` - Встроенный чат

## Модель монетизации

1. **Подписки для собственников:**
   - Бесплатный: ограниченный просмотр профилей
   - Базовый: неограниченный доступ к базе
   - Премиум: расширенные возможности

2. **Платные услуги для арендаторов:**
   - Верификация личности и дохода
   - Продвижение профиля в базе

## Разработка

Для разработки используйте:

```bash
# Запуск в режиме разработки
npm run dev

# Генерация Prisma клиента
npm run db:generate

# Применение изменений схемы БД
npm run db:push

# Открыть Prisma Studio
npm run db:studio
```

  - Хеширование паролей (bcrypt)
